# H₂O Thermo Calc

Веб-калькулятор теплофізичних властивостей води на основі апроксимаційних формул IAPWS-IF97 (рідка фаза, діапазон 0–370 °C, тиск 0.001–100 МПа).

## Структура проєкту

- `index.html` — розмітка сторінки
- `styles.css` — стилі
- `app.js` — UI-логіка калькулятора
- `water.js` — формули властивостей води
- `i18n.js` — переклади та логіка перемикання мов (UA / EN / PL / DE)
- `version.js` — константа `APP_VERSION`

## Запуск

Статичний сайт; обслуговується через `npx http-server` на порту 5000 (workflow `Start application`).

## Журнал змін

### v1.4.0
- Додано підтримку перегрітої водяної пари: режим (рідина / пара) визначається автоматично за тиском і температурою насичення.
- Властивості пари: густина — за рівнянням ідеального газу (R_s = 461.5 Дж/(кг·К)); cp, λ, μ — інженерні поліноми/лінійні наближення; ентальпія через теплоту пароутворення.
- Верхню межу температури підвищено з 370 °C до 800 °C.
- Виправлено переклад мітки фази "steam" з "Перегріта рідина" на "Перегріта пара" (та аналогічно для EN/PL/DE).
- Прибрано блокування при T > T_насич — тепер показуються властивості пари замість помилки.
- Поверхневий натяг для газової фази показується як «—» (не визначено).

### v1.3.0
- Додано поле введення тиску з вибором одиниць (МПа / кПа / бар / атм).
- Корекція густини за тиском через ізотермічну стисливість (≈4.5e-10 1/Па).
- Температура кипіння перераховується для заданого тиску (рівняння Антуана), фаза визначається відносно реального T насичення.
- Захист від параметрів за межами рідкої фази з відповідним повідомленням.
- Тиск показується в результатах, входить в історію та експорт CSV/PDF.
- Усі нові рядки перекладено всіма чотирма мовами.

### v1.2.0
- Історія останніх 10 розрахунків (зберігається в `localStorage`).
- Чипи з історії клацабельні — повторюють розрахунок одним кліком.
- Кнопка очищення історії.
- Дублікати в історії не множаться, а переміщуються нагору.

### v1.1.0
- Експорт результатів у CSV (з UTF-8 BOM для коректного відкриття в Excel).
- Експорт у PDF через системний діалог друку.
- Відображення версії додатку у футері.

### v1.0.0
- Базовий калькулятор: 11 теплофізичних властивостей рідкої води.
- Введення температури в °C / K / °F.
- Мультимовний інтерфейс: українська, англійська, польська, німецька.
- Вибір мови зберігається в `localStorage`, автовизначення з налаштувань браузера.
